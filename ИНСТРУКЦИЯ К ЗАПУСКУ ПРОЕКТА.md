# Инструкция локального запуска

Проект рассчитан на запуск через Docker Compose.

## Требования
- Docker Desktop (или Docker Engine + Compose plugin)

## Запуск через Docker Compose
1. В корне проекта выполните:
   ```bash
   docker compose up --build
   ```
2. Дождитесь запуска сервисов.
3. Откройте приложения:
   - Frontend: http://localhost:8080
   - Backend API docs: http://localhost:8000/docs

## Демо-пользователи
При `DEMO_SEED_ENABLED=true` (задано в `docker-compose.yml`) создаются демо-аккаунты:
- Работодатели:
  - `employer1@demo.example.com` / `employer123`
  - `employer2@demo.example.com` / `employer123`
  - `employer3@demo.example.com` / `employer123`
- Админ (demo): `admin@demo.example.com` / `admin123`
- Соискатель (demo): `seeker@demo.example.com` / `seeker123`

Админ также может быть создан из `.env` при старте:
- `ADMIN_EMAIL=admin@example.com`
- `ADMIN_PASSWORD=admin12345`

## Остановка
```bash
docker compose down
```
